# מדריך למשתמש — בדיקת סיכון בהריון

כלי להערכת סיכון בהריון, המיועד לאישה בהריון או המתכננת הריון. הכלי מעריך, על סמך
הפרטים שמוזנים, האם הם דומים למקרים שסווגו כ**הריון בסיכון גבוה**, ומציג את התוצאה בשפה ברורה.


## כניסה לאתר

- **הרצה מקומית** :
  ```bash
  pip install -r requirements.txt
  python webapp/app.py
  ```
  ואז לפתוח בדפדפן: `http://127.0.0.1:5000`.

---

## שימוש שלב-אחר-שלב

1. **מסך הפתיחה** — לחיצה על **"התחילי בדיקה"**.
2. **מילוי הטופס** — הטופס מחולק לארבע קבוצות:
   - *פרטים כלליים* — גיל, משקל, גובה (ה-BMI מחושב אוטומטית).
   - *מדדים רפואיים* — לחץ דם (סיסטולי/דיאסטולי), דופק, חום גוף (במעלות צלזיוס), רמת סוכר בדם.
   - *היסטוריה רפואית* — סיבוכים קודמים, סוכרת (לפני/בהריון), רקע נפשי.
   - *פרטי הריון ובדיקות* — מספר הריונות, חיסוני טטנוס, שבוע הריון, דופק עוברי, סוכר בשתן, VDRL, HBsAg.

   אין חובה למלא הכול — מלאי את מה שידוע לך. שדות כן/לא נבחרים מרשימה.
3. **קבלת הערכה** — לחיצה על **"קבלת הערכה"**. המערכת מזהה אוטומטית אילו נתונים הוזנו ובוחרת את
   ההערכה המתאימה: לפי מדדי בריאות, לפי בדיקות הריון, או הערכה משולבת של שתיהן.
4. **התוצאה** — מוצגת רמת הסיכון, אחוז, הסבר קצר, והמלצה להמשך. תחת **"מידע נוסף"** ניתן לראות את
   הפירוט לפי כל מודל.

---

## איך לפרש את התוצאה

- **"נמצא סיכון מוגבר"** — הפרטים דומים יחסית למקרים בסיכון. אין משמעות הדבר שקיים מצב רפואי; ההמלצה:
  **לפנות לצוות רפואי להמשך בדיקה והתייעצות.**
- **"לא נמצא סיכון מוגבר לפי הנתונים שהוזנו"** — אין דמיון גבוה למקרים בסיכון. זו אינה ערובה,
  והכלי אינו מחליף מעקב הריון שגרתי.
- **האחוז** מבטא את מידת הדמיון למקרים בסיכון מוגבר — ולא סבירות לקיומה של מחלה.

## מגבלות

- הכלי נבנה כפרויקט לימודי על בסיס מאגרי נתונים מוגבלים, ואינו בהכרח מייצג את כלל הנשים.
- ההערכה תלויה באיכות ובדיוק הפרטים שהוזנו; ייתכנו טעויות לכאן ולכאן.
- התוצאה אינה אבחון ואינה תחליף לבדיקות וייעוץ רפואי. בכל שאלה — פני לצוות רפואי מוסמך.
